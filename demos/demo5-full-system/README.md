## 功能整合清單

| 功能 | 來源 Demo | 整合方式 |
|------|-----------|---------|
| 人員偵測 + PPE 合規 | YOLO baseline | 即時 bbox overlay |
| 物理異常偵測 | Demo 2 | surprise 時間序列 + 警報 |
| 軌跡預測預警 | Demo 1 | 虛線軌跡 + 倒數計時 |
| YOLO/LeWM 雙重視野 | Demo 3 | 分欄顯示 + fusion score |
| PTZ 自主巡視 | Demo 4 | 多區域熱力圖 + 自動轉向 |
| 統計儀表板 | 新增 | 日報/週報/趨勢圖 |
| 警報歷史紀錄 | 新增 | 時間線 + 截圖回放 |

## Dashboard 頁面規劃

### 頁面 1: 即時監控 (Live)
- PTZ 即時畫面（含 YOLO bbox + LeWM 預測軌跡）
- surprise 即時曲線
- 警報 toast 通知

### 頁面 2: 區域總覽 (Overview)
- 風險熱力圖（各區域 surprise 均值著色）
- PTZ 當前朝向指示
- 各區域最後觀測時間

### 頁面 3: 警報紀錄 (Alerts)
- 時間線列表
- 每筆警報含：截圖、surprise 值、YOLO 偵測結果、觸發原因
- 篩選：日期、區域、嚴重度

### 頁面 4: 統計分析 (Analytics)
- 日/週/月 警報趨勢
- 高風險時段分析
- 預測命中率、誤報率追蹤
- 各區域風險排名
