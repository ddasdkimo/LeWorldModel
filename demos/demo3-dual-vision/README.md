# Demo 3: YOLO + LeWM 雙重視野

**技術論文素材** — 同一畫面展示 YOLO 的物件偵測與 LeWM 的物理理解如何互補。

## 展示效果

```
│            YOLO 偵測              LeWM 理解      │
│  ┌────────────────────┐  ┌──────────────────┐ │

```

## 核心價值：互補性分析

### YOLO 強、LeWM 弱的場景
- PPE 合規偵測（安全帽、反光背心）→ YOLO 擅長的分類任務
- 已知類別的物件識別 → YOLO 有明確的 class label

### LeWM 強、YOLO 弱的場景
- 未知類型的物理異常 → LeWM 不需要預定義類別
- 軌跡預測與危險預警 → YOLO 只看當下，LeWM 看未來
- 物理 vs 視覺區分 → LeWM 懂物理規律

### 兩者結合的協同效應
- YOLO 偵測到人 + LeWM 預測軌跡 = 精確的危險預警
- YOLO 偵測到物件 + LeWM surprise 高 = 物件狀態異常
- YOLO 無偵測 + LeWM surprise 高 = 未知物件/事件異常
