# 可行性分析

## 方向一：PTZ + YOLO + LeWM 工業安全偵測

### 結論：高度可行，建議優先推進

**差異化價值**：目前無人將 JEPA 世界模型應用於工業安全監控，研究空白明確。

### LeWM 補強的 4 個環節

| 環節 | YOLO 單獨 | + LeWM |
|------|----------|--------|
| 軌跡預測 | ❌ 只看當下 | ✅ 預測 2-5 秒後位置 |
| 物理異常偵測 | ❌ 只認訓練類別 | ✅ 偵測任何物理違規 |
| PTZ 智慧控制 | ❌ 固定巡邏 | ✅ 根據風險自主轉向 |
| 時序上下文 | ❌ 逐幀獨立 | ✅ 理解動作序列語義 |

### 延遲預算

| 模組 | 延遲 |
|------|------|
| YOLO v11 | 5-15ms |
| LeWM 推論 | 2-8ms |
| 決策融合 | 1-2ms |
| PTZ 控制 | 5-10ms |
| **總計** | **13-35ms (28-77 FPS)** |

### 硬體成本

MVP 配置：$630-1,030 USD

## 方向二：Reachy Mini + LeWM

### 結論：不可行（需換平台）

| 問題 | 說明 |
|------|------|
| 無手臂 | Reachy Mini 只有頭部 6DOF + 底座旋轉 |
| 算力不足 | RPi CM4 (~20 GFLOPS) vs CEM 需求 (~90 TFLOPS) = 差距 4,500× |
| 無 GPU | CM4 無 CUDA 核心 |

### 替代方案

- Reachy 2 ($70,000) + 外接 GPU → 完整可行
- 其他有臂機器人 + Jetson Orin → 精簡 CEM 可行
- Reachy Mini 僅用 LeWM encoder 做視覺特徵提取 → 價值有限
